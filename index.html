About

There is a common type of synesthesia in which people perceive colours when they see certain letters or numbers or combinations of letters or numbers.

This is a basic tool to demonstrate which colours four certain people (Yachen, Surabit, Fiona and Su) see when reading text like this.

Click these buttons to switch between people! (it might take a few seconds BE PATIENT!!!!)

Options

Background


You can use HEX colours (e.g. #F00 or #FF0000) or regular colour names (e.g. red).
default: #aaa (light grey)

"Pop" effects

You can select multiple effects. I don't recommend using both the outline and black overlay options.

You can use HEX colours (e.g. #F00 or #FF0000) or regular colour names (e.g. red).
default: #aaa (light grey)

Blur size is an integer (and measured in pixels).
default: 3px

Opacity is a number between 0 and 1, where 0 is totally transparent and 1 is completely opaque.
default: 0.5
